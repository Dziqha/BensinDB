### Examples
```go
// Create
BUAT TANGKI pengguna (id INT, nama TEKS, umur INT)

// Insert
ISI KE pengguna (1, 'Budi', 25)

// Select
PILIH * DARI pengguna DIMANA umur > 20

// Update
ATUR pengguna SET nama = 'Budi Santoso' DIMANA id = 1

// Delete
BAKAR DARI pengguna DIMANA id = 2

// Join
GABUNG pengguna DAN produk MENJADI transaksi DIMANA pengguna.id = produk.id

// Union (Style 1)
CAMPUR TANGKI tangki_a + tangki_b MENJADI hasil

// Union (Style 2)
SATUKAN tangki_a, tangki_b MENJADI hasil

// Order
URUTKAN TANGKI pengguna BERDASARKAN umur MENURUN

// Group
GRUPKAN TANGKI produk BERDASARKAN kategori SUM(harga)
```

### Known Limitations
- In-memory only (no persistence yet)
- Single-threaded execution
- No transaction support
- Basic WHERE conditions only (no nested conditions yet)
- Single column JOIN only
